# 🚀 Namka Control

> AI-Supported Project Management Dashboard

## 📊 Overview

**Namka Control** is a streamlined project management dashboard for Ali Mora's development projects. It provides real-time visibility into project status, priorities, and AI agent assignments — ensuring seamless coordination between human oversight and AI-assisted development.

- **Owner**: Ali Mora
- **Location**: Johannesburg, ZA
- **Updated**: 2026-03-29

---

## ✨ Features

### 📄 Single Source of Truth
- **Master.md Driven**: All project data lives in `Master.md`
- **Version Controlled**: Full git history of project evolution
- **AI Agent Collaboration**: Agents read/write reviews directly to Master.md

### 🤖 Multi-Agent Coordination Protocol (MACP)
- **Agent Reviews**: Each AI agent documents analysis and recommendations
- **Cross-Checking**: Agents review each other's work to reduce hallucinations
- **Consensus Building**: MACP workflow ensures quality through collaborative validation

### 📊 Real-Time Dashboard
- **Project Cards**: Visual overview of all active projects
- **Priority Tracking**: Color-coded priority levels (High/Medium/Low)
- **Status Monitoring**: Track Active, Blocked, or Complete status
- **Agent Assignments**: See which AI model is working on each project

### 🔗 GitHub Integration
- **Live Data**: Fetches Master.md directly from GitHub API
- **Always Current**: No sync lag or stale data
- **Commit History**: Track all changes with full git history

1. **Update Master.md**: Add/update project status
2. **Agent Review**: AI agent analyzes and documents findings
3. **Cross-Check**: Another agent validates the review
4. **Consensus**: MACP protocol ensures quality
5. **Implementation**: Execute with human approval
